# Report Generator

Утилита для обработки CSV файлов и генерации отчетов о производительности сотрудников.

## Описание

Скрипт `report_gen.py` анализирует CSV файлы с данными о сотрудниках и генерирует сводные отчеты о производительности по должностям.

## Установка

1. Клонировать репозиторий или скачать файлы проекта
2. Установить зависимости:

```bash
pip install -r requirements.txt

## Примеры использования

```bash
# Обработка одного файла
python report_gen.py --files data.csv --report performance

# Обработка нескольких файлов
python report_gen.py --files data1.csv data2.csv data3.csv --report performance

## Запуск тестов

Для запуска тестов используется pytest:

```bash
pytest test_report_gen.py -v

## Добавление новых отчетов

Для добавления нового генератора отчета требуется добавить его функцию, по аналогии с generate_performance_report. Функция отвечает за обработку списка словарей всех строк, полученных из всех csv-файлов. Таким образом, достаточно сгруппировать данные в соответствии с требованиями нового отчета, либо обработать их другим способом. В этой же функции необходимо отобразить
отчет в консоли.
